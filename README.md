## Descrição do projeto
Este projeto é um estudo de movimentação coletiva (Flocking) feito na Unity

## Funcionalidades ⚙️
✔️ Defina a quantidade de Peixes no seu cardume

✔️ Altere a velocidade deles

✔️ Defina a distância entre eles
